# Cursor Instructions for marcjoy.com Rebuild

## CONTEXT

This is the portfolio website for Marc Joy Media, an Afrofuturist
multimedia studio in Seattle founded by MarcJoy (Marcus Joyner).
The codebase was generated by Google Stitch and needs significant
corrections before it can go live. The site is a React + Vite +
Tailwind CSS + Framer Motion single-page application.

## CRITICAL RULES

1. ALL human figures, character portraits, and placeholder people
   must be BLACK. Dark brown to deep brown skin tones. Natural Black
   hair textures. Zero exceptions. Any Stitch-generated placeholder
   portraits of non-Black people must be replaced.

2. ALL image URLs currently point to `lh3.googleusercontent.com/aida-public/`
   which are temporary Stitch CDN links. These will ALL be replaced with
   Cloudflare R2 URLs once assets are uploaded. For now, use placeholder
   comments like `{/* TODO: Replace with R2 URL */}` where new images go.

3. The visual style is Nilepunk Afrofuturist: dark theme, cinematic,
   realistic (NOT illustrated or cel-shaded). Core palette:
   - Background: #0A0A0F / #13131b (current)
   - Teal accent: #2DD4BF (primary action color, ALL buttons)
   - Coral accent: #E07A5F (emphasis only, NEVER on buttons)
   - Indigo accent: #6366F1 (depth/secondary)
   - Text: #F5F0E8 (warm cream)
   - NO gold or yellow dominance

4. NO em dashes in any text content.

## PHASE 1: STRUCTURAL FIXES (Do First)

The site currently uses React state to switch pages. This means:
- No shareable URLs for inner pages
- Browser back button doesn't work
- No SEO for individual pages

```
npm install react-router-dom
```

- / = HomePage

- /work = WorkPage

## PHASE 4: ENHANCED ANIMATIONS (Do After Content is Fixed)

The Framer Motion setup is already imported. Add/improve:

### Scroll-triggered section reveals
```typescript
// Add to each section wrapper
<motion.section
  initial={{ opacity: 0, y: 40 }}
  whileInView={{ opacity: 1, y: 0 }}
  viewport={{ once: true, margin: "-100px" }}
  transition={{ duration: 0.6, ease: [0.22, 1, 0.36, 1] }}
>
```

### Hero parallax
```typescript
import { useScroll, useTransform } from 'motion/react';

// In hero component
const { scrollY } = useScroll();
const heroY = useTransform(scrollY, [0, 500], [0, 150]);
const heroOpacity = useTransform(scrollY, [0, 400], [1, 0]);

// Apply to hero image
<motion.img style={{ y: heroY, opacity: heroOpacity }} ... />
```

### Nav background on scroll
```typescript
const [scrolled, setScrolled] = useState(false);
useEffect(() => {
  const handler = () => setScrolled(window.scrollY > 50);
  window.addEventListener('scroll', handler);
  return () => window.removeEventListener('scroll', handler);
}, []);
// Then on nav: className={scrolled ? 'bg-background/90 backdrop-blur-md' : 'bg-transparent'}
```
